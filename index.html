Assignment 2B: One Scatter Plot and Two Datasets

This scatterplot illustrates the relationship between prostitution and vehicle theft in each district of San Fransisco.
The x-axis corresponds to the number of prostitution incidents and the y-axis corresponds the number of vehicle thefts.
The radius of the scatters are proportional to the total number of crimes in that district.

The two buttons allows the user to switch between two sets of data: "crimes of 2003" and "crimes of 2015".
To differentiate between the districs each scatter/circle is appended a tooltip. Simply hover the mouse over the circle to see the district label.

Notice that the axes does not change when changing data set. This is an intentional feature of the chart designed to better visualize change.
If the axes scale accordingly to the values it can be very difficult to see if a value increases or decreases.

Assignment 2D: Visualizing Geodata

The last exercise concerns visualizing geodata. The map contains every reported incedent of prostitution in San Francisco.
Using a K nearest neighbours (KNN) algorithm the incidents (or point on the map) is assigned to a cluster.
Using the 5 buttons the user can toogle between the number of clusters and see how the number of clusters affect the result.
