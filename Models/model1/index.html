Get To Know The Data!

Using the javascript library D3 (data driven documents) it was possible to create an interactive map of NYC. The map gives an overview of the constructed dataset and the extracted feaures. Using the three buttons the user can toogle between three different modes.

  • Overview - Shows the 5 boroughs of NYC
  • Income - Shows the variabilty of average income in NYC
  • Health - Shows variability in heart failure occurances in NYC

To the left of the map is a table showing additinal information about each area when hovered. This makes the dataset very intuitive and easy to explore. To see it for yourself jump to the next slide.

Fun fact: The average income of zip-code 10005 (location of Wall street) is almost 1.000.000 $.
