Decicsion Tree

The question is; can the borough of an observation be predicted based on the economical and health related features? From the exploration of the data, clearly large diffences exists between the boroughs of New York City. But is the variation between the boroughs disquisable from the overall variation? To explore this, a decision tree has been trained, using all features as input and the boroughs as targets.
In the next slide the decision tree has been visualized. By clicking on the nodes it is possible to expand or hide the branches of the tree in order to see how the decision tree performs the classification.
The tree uses the Gini impurity measure and has a max depth of 5 nodes. The performance of the classifier was evaluated using a 10-fold crossvalidation, and the following results were obtained.
Traning error = 74.4 %
Test error = 45.2 %

The max depth of 5 nodes was chosen to limit the amount of overfitting.
