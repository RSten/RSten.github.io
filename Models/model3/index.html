Decicsion Tree

As seen in on previous slide, large diffences exists between the boroughs of New York. The question is, if the differences between the boroughs are so large that we can predict which borough the zip-code belongs to based on the features presented in the previous slide. To do so, a decision tree has been trained, using all features as input and the boroughs as targets.
In the next slide the decision tree has been visualized. By clicking on the nodes it is possible to expand or hide the branches of the tree in order to see how the decision tree performs the classification.
Using Gini impurity measure and a max depth of 5 nodes the decision tree gave the following results:
Traning error = 74.4 %
Test error = 45.2 % - Using a 10-fold crossvalidation.

The max depth of 5 nodes was chosen to avoid to much overfitting.
