02806: Obesity and Lifestyle Related Diseases in NYC

This webpage along with the explanatory iPython notebook are the final product of the course 02806 Social Data Analysis and Visualization. The focus of this project is about the wast growing problem of obesity in the western world along with the lifestyle related diseases that often follows. Using features from multiple datasets (see list of datasets) it was possible to construct a dataset containing information about both the number of fastfood restaurants and the socio-econnomic aspects of each ZIP code (area) in NYC.

The setup of the webpage uses iFrames to make a diashow-like experience. Using the arrows in the side of the screen the user can swich between the different visualization.
